HTMLタグ備忘録

別言語とか使ってると意外とtableってどんな構成だっけ...画像ってどうやって出すっけ...ってなったので パッとみてコレダ！ってなるようにしました。

ページレイアウト系

記述タグ 実際の表示 説明 入れられるプロパティ
<p></p>

pタグ

 段落。テキスト入れたいなら適当にコレ。alignで文字寄せができる。 align = "方向"
<h1></h1>

h1タグ

 見出し文字。見出し用のテキストや適当にデカくしたい時にコレ。1のところを2~5に変更することで大きさなどを変更できる。
<hr></hr>
hrタグ		 横罫線を引く。プロパティで色々調整はできるが別にCSSで事足りると思う。idやclassをごっちゃにしたくない時に
  • width = "線の長さ"
  • size = "線の太さ"
  • align = "方向"
  • noshade 平面的に表示
<br> 改
行		 改行！
以上！		 clear = "画像に対しての回り込み解除方向"
<div></div>
divタグ
divタグ2
 分割用。エリアごとに分けたりidなどを指定してまとめて対象にしたい時とかに使う。コンテンツを分割するのにも使える。 align = "方向"
<pre></pre> 
preタグ
(br未使用で改行)
 整形済みテキスト。改行やタブをそのまま表示するのでbrいちいち書くのめんどくさいな...ってなった時に使う。
<address></address>
addressタグ
 アドレス情報。イタリック体で表示されるらしいがよくわからない。とりあえずアドレスはこれで囲めばOK
<blockquote></blockquote>
blockquoteタグ
 ブロック引用。上下左右に空きを作るがこれもよくわからない。使い道もよくわからない。
<a></a> aタグ ハイパーリンク。リンクを貼る時に使う。リンクは同じサイト内でも別のサイトに飛ばすのにも使える。
  • href = "URLやページ内のidなど"
  • name = "設定したいアンカーの名前"
